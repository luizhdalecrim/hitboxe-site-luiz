## 📋 Sobre o Projeto

O **Hitboxe** é um site institucional fictício para uma academia de boxe, criado pelo **Grupo 7 (Turma A)** da disciplina de **PCW (Padrões para Conteúdos Web)** e **PI (Projeto Integrador)**, no **1º Termo do curso de Análise e Desenvolvimento de Sistemas (ADS)** da **FIPP/Unoeste — 2026**.

O projeto simula um produto real desenvolvido pela empresa fictícia **NeoSys Consultoria de TI**, exercitando na prática os conteúdos de desenvolvimento front-end com HTML e CSS.

---

## 🌐 Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial — hero, diferenciais e prévia dos planos |
| `sobre.html` | Página institucional "Sobre a Hitboxe" |
| `Planos.html` | Catálogo de planos disponíveis |
| `plano-a.html` | Detalhes de um plano específico |
| `plano-treinos.html` | Plano de treinos do aluno |
| `login.html` | Tela de login |
| `cadastro.html` | Formulário de cadastro de novos alunos |
| `dashboard.html` | Painel do aluno logado |
| `consultoria.html` | Página de consultoria personalizada |
| `faq.html` | Perguntas frequentes |

---

## ✨ Funcionalidades

- **Hero section** com chamada para ação ("Iniciar Evolução")
- **Seção de diferenciais** da academia (planos personalizados, acompanhamento de evolução, histórico de frequência, consultoria)
- **Três planos de assinatura**: Round, Professional e Nocaute
- **Sistema de navegação** entre páginas (menu responsivo)
- **Telas de login e cadastro** para alunos
- **Dashboard** do aluno com gráficos de progresso
- **Consultoria** com profissionais da área
- **FAQ** com dúvidas frequentes
- **Unidades** em São Paulo e Bauru

## 🚀 Como Executar Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/gp7-neosys/hitboxe-site.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd hitboxe-site
   ```

3. **Abra o arquivo principal no seu navegador:**
   ```bash
   # No Linux/macOS
   open index.html

   # No Windows
   start index.html
   ```

Não é necessária nenhuma instalação adicional — basta abrir o `index.html` em qualquer navegador moderno.
